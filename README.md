Bu proje, 2025-2026 Güz Dönemi Web Programlama dersi kapsamında geliştirilmiştir. Projenin amacı, ASP.NET Core MVC teknolojisi kullanılarak gerçek hayatta kullanılabilecek bir Spor Salonu Yönetim ve Randevu Sistemi geliştirmektir.
Geliştirilen sistem sayesinde spor salonları;
•	sundukları hizmetleri,
•	antrenör bilgilerini,
•	antrenörlerin müsaitlik saatlerini,
•	üye randevularını
•	ve yapay zekâ destekli diyet önerilerini tek bir platform üzerinden yönetebilecektir.
Uygulama, rol bazlı yetkilendirme, CRUD işlemleri, REST API kullanımı, LINQ sorguları, veri doğrulama ve yapay zekâ entegrasyonu gibi Web Programlama dersinde öğrenilen tüm temel konuları kapsayacak şekilde tasarlanmıştır.
